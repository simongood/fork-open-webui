# 模擬 webui 實現與 Chatgpt 類似的前端頁面

# 開發工具
[nvm-windows](https://github.com/coreybutler/nvm-windows/releases) 下載 1.2.2

### node 環境安裝
    - `nvm install 20.19.0`
    - `nvm use 20.19.0`

### 專案環境安裝
    - `npm install`
    - `npm run dev`
